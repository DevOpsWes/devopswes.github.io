## Hey, I'm Wes 👋

I'm an engineer obsessed with building systems that work *while I sleep* — automating the toil, orchestrating the complexity, and making remote work feel like a superpower.

I've spent years at the intersection of **AI**, **DevOps**, and **automation**, turning painful manual processes into elegant, scalable pipelines. Whether it's zero-downtime deployments, LLM-powered workflows, or the perfect home-office setup, I care about the craft.

This blog is my notebook — ideas I'm testing, lessons from the trenches, and tools worth knowing.

- **DevOps** — CI/CD, Kubernetes, infrastructure-as-code, GitHub Actions, GitOps, and platform engineering

- **Virtual Desktop Environments** — VDI, cloud desktops, remote access solutions, and tools for building secure, high-performance virtual workspaces

## The Philosophy

> Automate the ordinary. Amplify the extraordinary.

If a machine can do it twice, write a script. If a script runs daily, containerize it. If a container matters, monitor it. And always leave time for the work that *only humans can do*.
